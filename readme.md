# Interpreter

Делаем примитивный интерпретатор.

## Типы данных

- Натуральное число: строка, состоящая только из цифр, образующих число.
- Строка: строка, начинающаяся с буквы.
- Список: несколько значений, разделенных пробелами и обрамленные квадратными скобками.

## Команды

### `SET`

Имеет синтаксис `SET <name> <value>`. Создает или перезаписывает переменную с именем `<name>` и значением `<value>`.

### `PRINT`

Имеет синтаксис `PRINT <name>`. Выводит значение переменной с именем `<name>`.

### `ADD`

Имеет синтаксис `ADD <name> <value>`. Добавляет значение `<value>` к переменной с именем `<name>`, если это возможно с точки зрения типов.
- Число может быть увеличено только на число.
- Строка может быть увеличена на строку (конкатенация) или на число, которое будет конвертировано в строку.
- Список может быть увеличен на список (конкатенация) или на число (добавление).

Уже реализована работа с числами. Нужно добавить работу с строками и списками. Подробнее в тестах.
